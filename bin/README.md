# `op-challenger`

The binary for the Optimism challenge agent.

## Usage

```text
Options:
  -v, --v...
          Verbosity level (0-4) [env: VERBOSITY]
  -w, --ws-endpoint <WS_ENDPOINT>
          The Websocket RPC endpoint used to index and send transactions. [env: OP_CHALLENGER_WS]
  -d, --dispute-game-factory <DISPUTE_GAME_FACTORY>
          The address of the dispute game factory contract. [env: OP_CHALLENGER_DGF]
  -l, --l2-output-oracle <L2_OUTPUT_ORACLE>
          The address of the L2OutputOracle contract. [env: OP_CHALLENGER_L2OO]
  -h, --help
          Print help
  -V, --version
          Print version
```
